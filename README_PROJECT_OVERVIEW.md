Project Overview by Vignesh V 22BME1206

- Frontend (React + Vite): located in /frontend
  - Dev: `npm run dev` (Vite)
  - Build: `npm run build`

- Backend (Node + Express): located in /server
  - Start: `node index.js`

- Environment: Copy .env.example -> .env and set OPENAI_API_KEY

- Docker: `docker-compose up --build` will bring up both services.
